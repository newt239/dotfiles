# dotfiles

First, run the following command to clone the repository.

```bash
xcode-select --install
git clone
```

## installation

```bash
make
```

## memo

```bash
git update-index --chmod=+x .bin/*.sh
```

## 反映されていない設定

- システム設定
  - キーボード
    - キーボードの輝度: 0
    - 🌐 キーを押して: 何もしない
    - テキスト入力
      - 入力ソース: Google 日本語入力を選択
  - トラックパッド
    - その他のジェスチャ
      - アプリ Expose: 3 本指で下にスワイプ
